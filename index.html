Kinect2Scratch4Mac

The bridge application to connect "KINECT" to Scratch on Mac OSX

About

This is Mac version of an application called Kinect2Scratch, which is developed by Stephen Howell. It sends data from Microsoft Kinect controller to Scratch, which is the programming environment developed by MIT Media Laboratory.

Anyway...
You can use Kinect from Scratch on Mac!!!!

Video

Getting Started

Required

  • Kinect
  • Mac OSX(10.6.8~)

    • I don't check whether run this app on 10.5 or not.

  • Xcode(4.x)

    • You can get it from here.

How to build

  1. Download a package of Kinect2Scratch4Mac
  2. Unpack it, and run "build.app"

    3. [!] Don't move an app from bin folder. Please use alias as Application. If relation of place of Application is changed, the app won't run.

How to use

First, download project for test and open it. Next, turn on "remote sensor connections" from Sensing>Slider sensor value(Right Click) in Scratch. After that, connect Kinect to your Mac, and run this app. Finally, pose this in front of Kinect. You can get data from kinect on Scratch.
